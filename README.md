# Rich-texteditor

Clone project to your local machine

Run script
>npm i

Commands to build project

- In development mode
  > npm run start
- In production mode (available under /build)
  > npm run build

Command to start project with HMR on a dev server (default http://localhost:8080)
  > npm run dev
